# windows broker

标签（空格分隔）： broker

---

## 安装
 1. 复制到要部署的目录，比如e:/inspireso/bin
 2. 解压，比如e:/inspireso/bin/broker
 3. 用管理员身份运行 install.bat
 4. 用管理员身份 start.bat

## 防火墙配置
1.允许内网访问9999端口
 
## 使用
启动服务: http://xxx.xxx.xxx.xxx:9999/net/service_name/start
停止服务: http://xxx.xxx.xxx.xxx:9999/net/service_name/stop
